# Inventory-Manager-With-C
A console-based inventory management system written in C that manages a list of products. It provides a user-friendly interface for adding, removing, and updating items, as well as searching for items by name, displaying inventory, sorting inventory by price, saving and loading inventory to a file, displaying low stock items, and displaying inventory summary.

## Features
- Add, remove, and update items
- Search for items by name
- Display inventory
- Sort inventory by price
- Save and load inventory to a file
- Display low stock items
- Display inventory summary

## Installation

# Linux
First clone the repository.
```bash
    git clone https://github.com/samsul-arafin/Inventory-Manager-With-C.git
```

Then navigate to the directory.
```bash
    cd Inventory-Manager-With-C
```

Compile the program.
```bash
    gcc -o main main.c
```

Run the program.
```bash
    ./main
```
